- **유형:** Labeling + Similar Public Implementations + Existing Data
- **분량:** 보고서(PDF) + 라벨링 결과(엑셀·구글시트)

## 목적

V1이 규칙 기반인데, 잔소리 선택은 4축(단계·카테고리·모드·디데이) 공간의 최적점 탐색 문제다. 256 조합 전부 설계 불가 → 대표 조합 직접 라벨링으로 "이 상태엔 이 스타일이 맞다"의 판단 기준 명시화.

## 제출 항목

1. 프로젝트 주제 + 과제 목적(우리 맥락)
2. Labeling Examples (구글시트 링크 + 라벨링 후 발견 패턴 + 애매·어려웠던 점)
3. Similar Public Implementations (2개 이상, 링크·유사점·참고점)
4. Existing Data and Code (데이터셋 이름·출처·Input/Output·활용 가능성)
5. 종합 정리 (초기 휴리스틱 / 예상 challenge / 접근 방법 유지 여부 / 추가 조사 필요 항목)
