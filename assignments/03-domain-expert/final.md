# 0416 도메인 전문가를 활용한 프로젝트 맥락 이해 — 최종

> 원본 제출 PDF: `course_materials/제출본_0416_도메인전문가.pdf` (개선 버전 + 초안 버전 동봉). 본 파일은 **개선 버전** 내용을 마크다운으로 옮긴 것.

## 1. 팀 프로젝트 주제

집중 세션 중 딴짓을 자동 감지하고 잔소리로 즉시 개입하는 자기조절 앱 **Zan Sol**.
Product goal: "사용자가 딴짓 순간에 집중 과업으로 복귀하도록 돕는다."

## 2. 도메인 전문가 후보

| 전문가 | 우리 프로젝트에 주는 지식 |
|---|---|
| ① 경희대 교수학습지원센터 학습상담사 | 학생 도피 행동에 대한 **이론적 개입 절차**. "어떤 순서로 개입하면 돌아오는가"의 판단 체계 |
| ② 고등학교 담임교사 / 자습감독 교사 | 현장에서 매일 학생의 폰 사용·졸음·도피를 잡아온 **실전 경험**. "어떤 말이 먹히고 어떤 말에 더 반항하는가"의 체화된 판단 |
| ③ 자기조절 잘하는 경희대 4.0+ 선배 | "관찰자 없이 혼자 공부할 때 폰 유혹을 어떻게 이기는가"의 **실제 행동 규칙**. 시스템이 대체해야 할 자기통제 패턴 |

학습상담사 = 이론(왜 이렇게), 교사 = 실전(뭐라 해야), 선배 = 역방향(도피 안 하는 사람의 규칙).

## 3. 전문가로부터 얻고 싶은 핵심 지식

가장 중요한 단일 질문: **"도피 중인 학생에게 개입했을 때, 실제로 돌아오게 만든 경험이 있는가? 그때 무엇이 먹혔는가?"**

세부 지식:
- **개입 순서:** 공감부터인가, 압박부터인가? 순서가 결과에 얼마나 영향을 주는가?
- **역효과 임계점:** 어떤 상황에서 개입이 오히려 반발을 불러오는가? 멈춰야 할 시점은?
- **복귀 행동:** "공부해" 같은 추상적 지시 대신, 실제로 몸이 움직이게 만든 구체적 제안은? (예: "일단 노트 펴봐")
- **에스컬레이션 파라미터:** 5분/15분/30분은 설문 분포 기반 초기값. 전문가 인터뷰와 출시 후 복귀율 데이터로 조정 예정.

## 4. 지식 확보 방법

| 전문가 | 방법 | 비고 |
|---|---|---|
| 학습상담사 | 비구조화 인터뷰 30분 | 교수학습지원센터 통해 접근 |
| 고등학교 교사 | 비구조화 인터뷰 30분 (팀 네트워크) | 야자 감독 경험 있는 교사 우선 |
| 4.0+ 선배 | 비구조화 인터뷰 각 20분 × 3~5명 | 팀 네트워크로 확보 |
| 보조 자료 | 학술 논문 (Steel & König 2006, Zimmerman 2000) | task aversion, 자기조절 이론 프레임 |

## 5. 간단한 heuristic (V1 핵심 루프)

학습상담사와 고등학교 교사의 개입 절차를 종합한 판단 흐름. **V1은 이 사람의 판단 절차를 규칙으로 옮긴 것.** 에스컬레이션 간격 5/15/30분은 자체 설문(경희대 34명) "딴짓 지속 10~30분" 응답 분포의 초기·중반·상한.

```
사용자가 앱 전환
  ↓
딴짓 리스트에 있는가? — No → 세션 계속
  ↓ Yes
[5분] 공감 개입 — "막막하지? 자료 1개만 찾아보자"
  ↓
복귀? — Yes → 세션 계속
  ↓ No
[15분] 긁기 시작 — "15분만 더 하면 오늘 목표 달성"
  ↓
복귀? — Yes → 세션 계속
  ↓ No
[30분] 제대로 긁기 — "내일의 너한테 미안하지 않을까?"
  ↓
복귀? — Yes → 세션 계속
  ↓ No
미완료 → 다음 세션 난이도 하향
```

**사람의 판단이 개입되는 핵심 지점:** 각 단계에서 "이 사용자에게 어떤 스타일의 말이 먹히는가" 선택. V1은 일반화하여 랜덤 처리, V2에서 Contextual Bandit으로 자동화.

## 6. 예상되는 scope/challenge

- **핵심 도전 — 개인차:** "어떤 학생에게 어떤 말이 먹히는가"는 전문가도 시행착오로 파악. 규칙으로 일반화 한계가 있으며, 이것이 V2 개인화 ML이 필요한 근본 이유.
- **역효과 관리:** 잔소리 과잉 → 짜증 → 앱 삭제. 출시 후 D7 리텐션과 복귀율을 동시 모니터링.
- **인터뷰 접근:** 학습상담사는 교수학습지원센터 경유, 교사·선배는 팀 네트워크. 일정 조율은 필요하지만 확보 자체에 큰 어려움 없음.

## 후속 진화 (이 과제 이후 CONTEXT 발전)

이 과제 시점의 5/15/30분은 **딴짓 지속 시간 분포**에 맞춘 초기값이었다. 이후 03 Speed 과제·14 메타 레슨에서 프레이밍이 진화해 **즉시 / 30초 / 2분**(Nielsen 1초 + Siebers screenertia 30초 + Mark 40초의 3배)으로 단축됨. 근거 축이 *"딴짓이 지속되는 시간"* → *"사고 흐름을 끊을 수 있는 한계"* 로 reframe됨. 현재 CONTEXT 5-6에 최종본 기록.
