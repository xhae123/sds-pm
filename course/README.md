# 수업 프레임워크 요약 (우리 팀 관점)

> **출처 고지:** 이 폴더의 문서들은 수업 강의자료를 수강생(김우진)이 **개인 학습 목적**으로 정리한 2차 요약이다. 강의자료 원본은 저작권상 레포에 포함하지 않는다 (수업 지침 명시). 이 요약은 "우리 팀 프로젝트 의사결정에 쓰기 위한 체크리스트" 형태이며, 교수 슬라이드 원문을 대체하지 않는다.
>
> **최종 학습 기준은 항상 주 강의 교재와 실시간 강의 내용**이다. 이 문서가 기억나지 않거나 모순되면 강의 자료를 다시 확인할 것.

## 왜 이게 필요한가

우리 팀이 매번 과제를 받을 때마다 다시 꺼내봐야 하는 것은 두 가지다:

1. **수업 원칙과의 매핑** — 교수님은 "혁신적 결과"가 아니라 "수업에서 배운 걸 명시적으로 적용했는가"를 평가한다.
2. **과제별 제출 포맷** — 매 과제가 요구하는 필수 항목/분량/어조가 다르다.

이 폴더는 그 둘을 상시 참조할 수 있게 압축한 것이다.
