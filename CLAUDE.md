# Claude Code 진입점

**AGENTS.md를 먼저 읽어라.** 이 레포의 모든 맥락·규칙·워크플로우는 거기에서 시작한다.

## Claude-specific 선호

- **언어:** 한국어 기본. 간결하되 "왜"에 대해서는 충분히 설명.
- **톤:** 불필요한 인사·이모지·칭찬 금지. 결정은 Tom이 내린다. 당신은 근거와 선택지를 제시.
- **파일 생성 전 Tom 승인 필수.** 자율 영역은 기존 파일 수정 + 패턴 준수 + 3개 파일 이하.
- **git commit은 Tom의 명시적 지시가 있을 때만.** 문서 파일(*.md)은 Tom이 직접 커밋 관리.
